# Bitget Public API Endpoints (USDT-FUTURES)

Base URL:
- https://api.bitget.com

Endpoints used:
- Contracts: `GET /api/v2/mix/market/contracts?productType=USDT-FUTURES`
- Tickers: `GET /api/v2/mix/market/tickers?productType=USDT-FUTURES`
- Candles: `GET /api/v2/mix/market/candles?symbol=BTCUSDT&granularity=900&productType=USDT-FUTURES&limit=200`

Granularity (seconds):
- 15m = 900
- 1h = 3600
- 4h = 14400
